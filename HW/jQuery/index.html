How to Design a Magazine Spread

Do you have the creative talent for graphic design but don't know where to start? Here is a simple step-by-step tutorial on how to design a basic two-page magazine spread, keeping in mind spacing, typography, and color. Good luck!

Here are the InDesign download instructions:

Step 1 - Setting Up

Open up InDesign and create a 8.5x11 spread with 1 inch margins. You can change the size of the document at any point, but this is the standard size for starting out. InDesign will automatically ask you about document set up when you first open the application.

Step 2 - The Headline

Now that you've got the set up all done, start by being bold with the headline. You can always dress things down later on, but for now you want a headline that grabs the reader's attention. Mess around with fonts, sizing, and spacing to create something that will stand out.

Step 3 - Images

Add in strong images that tell a story. You need one "center of visual impact" or CVI on a page so decide if you'd like the image to be the first thing a reader sees and go for it. Down below, I wanted the image to be big enough that it draws the reader there first. Use images in different sizes and orientations to balance out the white space and eventually the text when we add that in later.

Step 4 - Text

Now that you've got some visual impact, add in some copy. I always stick to the widths of the columns laid out on the page. You want to make sure your text isn't hyphenated and is around a size 9pt. so that it is legible for a reader.

Step 5 - Extra Design Elements

At this point, your spread is looking great! Now, what you want to do is spice the page up by changing colors, adding shapes, and making the page a little more interesting. Maybe add a design element that ties into the image, or even an element that adds intrigue. This is the hardest step in the design process because less is more so try not to add design elements for no reason. Here, I have added sound-looking bars because the story is about music, and a few extra black lines to tie in the headline. Now, you're done!
